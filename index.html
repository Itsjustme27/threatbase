Integration

Quick Integration Examples

Drop HimalayaFeed into your existing security stack in under a minute.

🐧
Linux iptables / ipset
# Block all feed IPs using ipset ipset create himalayafeed hash:ip wget -qO- https://raw.githubusercontent.com/\ kalidada18/himalayafeed/main/malicious_ips.txt \ | grep -E '^[0-9]' \ | while read IP; do ipset add himalayafeed $IP done iptables -I INPUT -m set \ --match-set himalayafeed src -j DROP
🌐
NGINX Blocklist
# Generate NGINX deny config curl -fsSL https://raw.githubusercontent.com/\ kalidada18/himalayafeed/main/malicious_ips.txt \ | grep -E '^[0-9]' \ | sed 's/^/deny /; s/$/ ;/' \ > /etc/nginx/conf.d/himalayafeed.conf nginx -s reload
🐍
Python — Load & Check
import requests url = "https://raw.githubusercontent.com/\ kalidada18/himalayafeed/main/malicious_ips.txt" resp = requests.get(url) blocklist = { line for line in resp.text.splitlines() if line and not line.startswith('#') } def is_malicious(ip: str) -> bool: return ip in blocklist
⚡
Cloudflare Workers / Edge
// Fetch & cache the feed at the edge const resp = await fetch( "https://raw.githubusercontent.com/\ kalidada18/himalayafeed/main/malicious_ips.txt" ); const blocklist = new Set( (await resp.text()).split('\n') .filter(l => /^[0-9]/.test(l)) ); // In your request handler: if (blocklist.has(clientIP)) { return new Response('Forbidden', { status: 403 }); }
